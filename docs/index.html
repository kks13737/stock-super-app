나만의 주식관리 슈퍼앱 문서 요약

매매일지, 뉴스 수집, 공포탐욕지수를 하나의 대시보드에서 관리하는 개인용 투자 보조 앱

Python FastAPI + uv 정적 HTML/CSS/JS FE SQLite DB

PRD 핵심

  • 날짜별 주식 매매일지 캘린더
  • 네이버 증권 뉴스 자동/수동 수집
  • CNN 공포탐욕지수 자동/수동 수집
  • 중복 저장 방지 및 실패 로그

아키텍처 핵심

  • API, Service, Crawler, Scheduler 분리
  • 수집 로직은 BS4 전용 모듈에서 처리
  • DB는 매매일지, 기사, 지수, 작업 로그 저장
  • 모든 시간 기준은 KST로 통일

운영 핵심

  • 자동 수집과 수동 수집이 동일 로직을 사용
  • HTML 구조 변경 시 파서 모듈만 우선 수정
  • 수집 실패는 로그와 재시도로 대응
  • 시간대 오류는 서버 KST 설정으로 방지
